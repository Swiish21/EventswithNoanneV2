

**Events with Noanne v2**
==========================

**Website Template for Events Manager Professionals**
---------------------------------------------------

**Overview**
------------

Events with Noanne v2 is a modern and responsive website template designed specifically for events manager professionals. This template provides a clean and elegant design to showcase projects, services, and team members. With its multiple pages and features, this template is perfect for architects, interior designers, construction companies, real estate developers, and professionals looking to showcase their projects and services online.

**Features**
------------

* **Responsive Design**: Compatible with all devices and screen sizes
* **Multiple Pages**:
	+ Home
	+ About
	+ Services
	+ Projects
	+ Blog
	+ Blog Details
	+ Contact
* **Image Sliders and Carousels**: Showcase your projects and services with style
* **Animated Headlines**: Grab attention with animated headlines
* **Progress Bars**: Display progress and statistics in a visually appealing way
* **Contact Form with Validation**: Easy-to-use contact form with validation
* **Newsletter Subscription Form**: Stay in touch with your audience
* **Testimonials Section**: Showcase customer testimonials
* **Team Members Section**: Highlight your team members
* **Google Maps Integration**: Display your location with Google Maps
* **Social Media Links**: Connect with your audience on social media

**Technical Details**
--------------------

* **Programming Languages**: HTML, CSS, JavaScript, PHP
* **Frameworks and Libraries**:
	+ Bootstrap
	+ jQuery
	+ Owl Carousel
	+ SlickNav
	+ Magnific Popup
	+ FontAwesome
	+ Themify Icons
	+ Google Fonts

**Installation**
---------------

1. Download the template files
2. Extract the files to your desired directory
3. Open the `index.html` file in your web browser to view the template

**Customization**
----------------

* Edit the `style.css` file to customize the template's styles
* Edit the `script.js` file to customize the template's JavaScript functionality
* Edit the `index.html` file to customize the template's content
